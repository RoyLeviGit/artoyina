## Notes

- **Painter's Studio (67 MB):** The largest file in this episode. Download will take longer. Worth it — 16,951px wide means any detail zoom is flawless.
- **Stone Breakers:** The original was destroyed in the Dresden bombing (February 1945). This is the only version that exists — a pre-war photograph of the painting. Label it accordingly in the edit.
- **Origin of the World (33 MB, 11,000px):** Extremely explicit. Decision about how to use it in the edit is in production/shot-list.md. File is here and ready either way.
- **Burial at Ornans:** Only 2,777px tall for a 6,042px-wide painting — the physical canvas is very wide and short. Full-width crops will work beautifully; tall crops will be limited.
