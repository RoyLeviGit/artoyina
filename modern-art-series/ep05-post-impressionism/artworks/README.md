## Notes
- **La Grande Jatte:** 20,000px wide (~200MB+). Download separately with patience. Use detail crop for the dot zoom — export a section at full res.
- **Van Gogh Bedroom:** 15,016×11,741 from C2RMF (Orsay version). Large file — ~20–30MB JPEG. The Van Gogh Museum version (Amsterdam) is an alternate but the Orsay version is fine.
- **Hiroshige:** Used to illustrate Japanese print influence on Van Gogh's Paris period. Any of Hiroshige's Famous Views of Edo series would work.
- **Where Do We Come From?:** 3,090×1,175 captures the panoramic format well (painting is 139×375cm in real life). Pan slowly right-to-left in edit.
- **Wheat Field with Crows:** 3,757×1,764 — similar wide-panoramic format. Use at full width.
- **The Seurat dot-zoom graphic** should be built in AE using La Grande Jatte's high-res file — pull back from extreme close-up to reveal the full scene. See `production/shot-list.md`.
